# VitrinAI Frontend

Frontend do projeto VitrinAI construído com Nuxt 3, Vue 3 e Tailwind CSS.

## Tecnologias

- **Nuxt 3** - Framework Vue.js para aplicações modernas
- **Vue 3** - Framework JavaScript progressivo
- **Tailwind CSS** - Framework CSS utility-first

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse http://localhost:3000

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
